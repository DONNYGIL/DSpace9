## 📌 Descripción

Implementación, configuración y personalización de **DSpace 9** (repositorio institucional open source) en **AWS EC2** con **Docker**.

Basado en mi experiencia como **Consultor Técnico Senior** en LA Referencia / RedCLARA y en el **Instituto Nacional de Innovación Agraria (INIA)**.

---

## 🚀 Funcionalidades

- ✅ **Migración de DSpace 6/7 a 9** - Proceso completo de actualización
- ✅ **Despliegue en AWS EC2** - Infraestructura en la nube
- ✅ **Contenerización con Docker** - Facilita despliegues y escalabilidad
- ✅ **Personalización de UI con Angular** - Interfaz moderna y responsive
- ✅ **Implementación de DSpace-CRIS** - Sistema de gestión de investigación
- ✅ **Interoperabilidad OAI-PMH** - Cosecha de metadatos
- ✅ **Integración con OpenAIRE y La Referencia** - Visibilidad internacional
- ✅ **Backup y recuperación automatizada** - Scripts de respaldo

---

## 🛠️ Tecnologías utilizadas

| Área | Tecnologías |
|------|-------------|
| **Repositorio** | DSpace 9 |
| **Cloud** | AWS EC2, S3 (almacenamiento) |
| **Contenedores** | Docker, Docker Compose |
| **Backend** | Java 17, Spring Boot |
| **Frontend** | Angular 15, TypeScript |
| **Base de Datos** | PostgreSQL 13 |
| **Búsqueda** | Solr 8 |
| **Web Server** | Apache Tomcat 9, Nginx |
| **Autenticación** | Shibboleth, LDAP, OAuth2 |
